Detailed Description

Embedded Image Manipulation Library include file.

Include this file in your project to perform basic image manipulation functions, such as scaling, cropping, and color conversion.

Crops input image with x/y and width/height, copies to output image.

Crops input image at the given (x, y) coordinate. The width and height of the crop comes from the image_out struct parameters. The cropped image is copied to the image buffer in image_out. Note that the color spaces of image_in and image_out must match. Also, the function will fail (EIML_ERROR) if the crop extends beyond the width or height of image_in.

Parameters
image_inImage to be cropped
image_outCropped image
xWhere to crop (x) on input image
yWhere to crop (y) on input image
Returns
eiml_ret EIML_ERROR on error, EIML_OK otherwise
